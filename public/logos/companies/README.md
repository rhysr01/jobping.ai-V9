## Logo Requirements

- **Format**: SVG (preferred) or PNG
- **Size**: Optimized for 100px height display
- **Style**: Full color logos (no grayscale)
- **Background**: Transparent
- **Naming**: Lowercase, matches company name in `lib/companyLogos.ts`

## Where to Get Logos

- **Simple Icons**: https://simpleicons.org/ (free SVG downloads)
- **Company websites**: Official brand assets pages
- **Logo.dev API**: https://logo.dev/ (free API)

## Notes

- Only companies with logos in this directory will be displayed
- Broken/missing logos are automatically hidden (no text fallback)
- Logos are cached for 1 hour via API
- System automatically shows top 30 companies by match frequency
